# Claude Code のサインイン

ターミナルで `claude` を起動し、ブラウザ経由で認証コードを取り込む。

## 1. 認証方式を選択する

起動後に表示されるメニューで `1. Claude account with subscription`（Pro/Max/Team/Enterprise 契約）を選ぶ。

![認証方式の選択](../images/claude-code-login-method.png)

## 2. 「外部サイトを開く」ダイアログは閉じる

`claude.com` への遷移を促すダイアログが表示されるが、ここから `Open` を押すとリダイレクト先（ローカルホスト）の都合で正常に認証が完了しない。**このダイアログは閉じる**。

![外部サイトのダイアログを閉じる](../images/claude-code-open-external.png)

## 3. コンソール上の URL を `Ctrl + Click` で開く

コンソールに `Browser didn't open? Use the url below to sign in` と URL が表示される。その URL を `Ctrl + Click` で開き、ブラウザで Claude にログインする。

![コンソールの URL](../images/claude-code-login-url.png)

## 4. OAuth を承認する

ブラウザで `Claude Code さんが Claude chat account への接続を希望しています` という画面が出るので `承認する` をクリックする。

![OAuth 承認](../images/claude-code-authorize.png)

## 5. 認証コードをコピーする

承認後に表示される認証コードを `コピーしました` ボタンでコピーする。

![認証コード](../images/claude-code-auth-code.png)

## 6. コンソールにコードをペーストし、クリップボード参照を許可する

コンソールの `Paste code here if prompted >` に貼り付けると、ブラウザがクリップボード参照の許可を求めてくる。`許可` をクリックするとコードが貼り付けられ、サインインが完了する。

![クリップボード参照の許可](../images/claude-code-clipboard-permission.png)
